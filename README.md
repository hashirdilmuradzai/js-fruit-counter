# JavaScript Fruit Counter

A beginner JavaScript assignment that counts how many times a specific fruit appears in an array.

## Features

- Count occurrences of a fruit in an array
- Case-insensitive matching (e.g., `apple` and `Apple`)
- Display the total count
- Accept any fruit name (Bonus)
- Handle an empty array gracefully (Bonus)

## Example

```text
Input:
["mango", "apple", "watermelon", "apple", "grape", "Apple"]

Fruit:
apple

Output:
3
```

## Technologies Used

- HTML5
- JavaScript

## JavaScript Concepts Used

- Arrays
- Loops
- Variables
- `if` statements
- String methods
- Comments for code explanation

## Restrictions

- Do not use `Array.filter()`
- Do not use AI-generated full solutions

## Purpose

This project was created to practice array traversal, conditional logic, string comparison, and counting values in an array using JavaScript without relying on built-in array filtering methods.
